# movie_seat_booking-css-JavaScript-jQuery-JSON

Web based application developed using CSS, JavaScript (jQuery, JSON) that enables users to customize their seats in a movie theatre
Application is modelled on Dominion Cinema (First Class)
The application allows users to be able to select seats, undo that selection and when all seats have been selected, it would be possible to click ‘Continue’.
The application also provides a unique way of giving feedback to the user by considering details such as-Which seats are available? Which have been selected? What the user can do next? The seat prices are? 
Special attention and efforts are put to ensure that the application is aesthetically pleasing and logically accurate in addition to being user friendly.
